# Buyda v2.0.0 - 基于选品策略白皮书的完整选品评分系统

## 🚀 版本亮点

### 核心功能升级

**基于跨境电商选品策略白皮书方法论，实现完整的产品评分和市场分析体系**

### 新增功能

#### 1. 📊 产品综合评分卡（五维评估模型）

- **市场体量（30%权重）**：月搜索量/购买量评估
- **竞争强度（25%权重）**：广告竞品数/CPC/集中度评估
- **利润空间（20%权重）**：毛利率/利润率评估
- **趋势指数（15%权重）**：环比/同比增长率评估
- **供应链优势（10%权重）**：1688货源质量/交付周期评估

**评分等级：**
- ⭐⭐⭐ 综合评分 ≥80分 - 强烈推荐
- ⭐⭐ 综合评分 60-79分 - 可以尝试
- ⭐ 综合评分 40-59分 - 需谨慎
- ⛔ 综合评分 <40分 - 不建议进入

#### 2. 📈 品类竞争力雷达图（品类6力）

- **兴趣度**：基于搜索量趋势
- **市场体量**：月搜索量/购买量
- **垄断程度**：CR3占比（越低越好）
- **利润潜力**：平均利润率
- **新品机会**：新品占比
- **发展趋势**：环比/同比增长率

**可视化：** 六边形雷达图，直观展示品类竞争力

#### 3. 📊 市场体量分级器

**7个市场等级：**
- 🟢 冷门市场（0-3万月搜索量）- 机会型
- 🟡 小型市场（3-8万）- 探索型
- 🟡 中小型市场（8-15万）- 稳定型
- 🔵 中型市场（15-30万）- 成长型
- 🔵 中大型市场（30-50万）- 扩张型
- 🟠 大型市场（50-80万）- 激烈型
- 🔴 超红海（80万+）- 红海型

#### 4. 📈 趋势智能判定（三维度评估）

**趋势判定标准：**
- 环比增长 >15% → 🚀 上升趋势
- 同比增长 >20% → 📈 爆发趋势
- 新品占比 >30% → 🆕 创新趋势

**趋势等级：**
- 🚀🚀🚀 超热趋势（满足3项）
- 🚀🚀 热门趋势（满足2项）
- 🚀 上升趋势（满足1项）
- ➡️ 平稳趋势（无显著变化）
- ⚠️ 衰退趋势（有预警信号）

#### 5. ⚠️ 垄断风险评估器

**三维度评估：**
- **CR3 指标**：前3品牌销量占比（>60% 高风险）
- **广告密度**：广告竞品数/总竞品数（>40% 高风险）
- **价格集中度**：TOP10价格区间占比（>50% 高风险）

**风险等级：**
- 🔴 高风险：⛔ 不建议进入
- 🟠 中风险：⚠️ 谨慎进入，需差异化策略
- 🟢 低风险：✅ 可以进入

## 📦 安装方法

### 方法一：下载安装包

1. 下载 `buyda-v2.0.0.zip`
2. 解压得到 `dist/` 目录
3. 打开 Chrome → `chrome://extensions/`
4. 开启「开发者模式」
5. 点击「加载已解压的扩展程序」→ 选择 `dist/` 目录

### 方法二：从 GitHub Release 安装

1. 访问：https://github.com/aoshuang99-netizen/buyda/releases/tag/v2.0.0
2. 下载 `buyda-v2.0.0.zip`
3. 按照方法一安装

## 🛠 使用指南

### 产品评分功能

1. 打开亚马逊产品页面
2. 点击插件图标 → 选择「📊 评分」标签
3. 查看产品综合评分卡（五维评估）
4. 切换到「📈 雷达图」查看品类竞争力

### 市场分析功能

1. 打开亚马逊产品页面
2. 点击插件图标 → 选择「📈 市场」标签
3. 切换三个视图：
   - **市场体量**：查看市场等级
   - **趋势判定**：查看趋势信号
   - **垄断风险**：查看风险评估

## 📚 理论基础

### 跨境电商选品策略白皮书

本版本完整实现了白皮书中的核心方法论：

**三维势能模型：**
- 顺品类之势（品类生命周期、创新机会）
- 顺需求之势（消费者行为、痛点分级）
- 顺平台之势（流量规则、算法机制）

**六维竞争力模型：**
- 品类6力：兴趣/体量/垄断/利润/新品机会/趋势
- 产品6力：需求量/竞品数/投产/CPC/集中度/趋势

**五维评估模型：**
- 市场体量（30%）+ 竞争强度（25%）+ 利润空间（20%）+ 趋势指数（15%）+ 供应链优势（10%）

**选品漏斗模型：**
- 阶段一：初筛（10-30款，日销>0.5单，利润率>0%）
- 阶段二：精选（10款，Vine测评×2，日销>1单，利润率>5%）
- 阶段三：矩阵布局（站外放量、Vine测试、广告测试、价格测试）
