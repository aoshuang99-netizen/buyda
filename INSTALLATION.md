# 📦 AI 选品助手 - 安装与加载指南

## 方法一：开发者模式加载（推荐用于测试）

### 步骤 1：打开 Chrome 扩展页面

在 Chrome 浏览器地址栏输入：
```
chrome://extensions/
```

或通过菜单访问：
1. 点击浏览器右上角 **三个点** (⋮)
2. 选择 **扩展程序** (Extensions)
3. 或使用快捷键：`Cmd+Shift+E` (Mac) / `Ctrl+Shift+E` (Windows)

### 步骤 2：启用开发者模式

在扩展页面右上角，找到 **"开发者模式"** 开关并打开它。

### 步骤 3：加载插件

1. 点击左上角的 **"加载已解压的扩展程序"** 按钮（Load unpacked）
2. 在文件选择器中，导航到项目目录：
   ```
   /Users/frankzhao/WorkBuddy/20260327160327/ai-selection-extension/dist
   ```
3. 选择 `dist` 文件夹并确认

### 步骤 4：验证安装成功

成功加载后，您应该能看到：
- ✅ 插件卡片出现在扩展列表中
- ✅ 插件名称：**AI 选品助手**
- ✅ 图标显示在浏览器右上角（或扩展菜单中）
- ✅ ID 格式：`xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`

2. 找到 **"AI 选品助手"**

- 在 `chrome://extensions/` 页面
- 点击插件的 **刷新图标** 🔄
- 或按 `Cmd+R` / `Ctrl+R` 刷新插件

## ❌ 常见问题

### Q: 插件加载失败
**可能原因：**
- `dist` 文件夹不完整
- 未开启开发者模式
- 文件权限问题

**解决方法：**
```bash
cd /Users/frankzhao/WorkBuddy/20260327160327/ai-selection-extension
npm run build
```
然后重新加载插件。

### Q: 浮动面板不显示
**可能原因：**
- 不在亚马逊产品详情页
- 页面未完全加载
- 扩展无权限

**解决方法：**
1. 确保页面 URL 包含 `/dp/ASIN` 格式
2. 刷新页面
3. 检查扩展权限设置

### Q: 无法提取产品数据
**可能原因：**
- 亚马逊页面结构变化
- 不同站点（.com vs .cn）差异

**解决方法：**
1. 切换到 amazon.com 测试
2. 在 Console 查看错误信息
3. 查看源代码 `src/content/amazon.tsx` 中的选择器

### Q: 数据不准确
**说明：**
- 当前 MVP 使用实时页面提取
- 后续版本将集成 Keepa API 获取历史数据
