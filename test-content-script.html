1
检查插件是否加载

2
测试 Content Script 消息

3
手动注入 Content Script

❓ 故障排除

  • 如果插件未加载：刷新 chrome://extensions/ 页面
  • 如果 content script 未注入：关闭所有 Amazon 标签页，重新打开
  • 如果仍然失败：检查浏览器控制台（F12）的错误信息
