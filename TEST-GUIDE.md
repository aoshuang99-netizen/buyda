# Buyda 插件快速测试指南

## 🚀 快速开始

### 步骤 1：重新构建并加载插件

```bash
cd ai-selection-extension
./reload-extension.sh
```

或者手动执行：
```bash
npm run build
```

### 步骤 2：刷新浏览器插件

1. 打开 Chrome 浏览器
2. 访问：`chrome://extensions/`
3. 找到 "Buyda个性化选品智能体"
4. 点击刷新按钮 🔄

### 步骤 3：测试 Amazon 产品分析

访问任意 Amazon 产品页面，例如：
- https://www.amazon.com/dp/B07SQZWFTC (CRZ YOGA Leggings)
- https://www.amazon.com/dp/B08N5KWB9H (其他产品)

等待 2-3 秒，右侧应该自动显示 **"Buyda 智能分析"** 浮动面板。

## 🐛 常见问题排查

### 问题 1：插件加载失败
**症状**：`chrome://extensions/` 中显示错误

**解决方案**：
1. 检查 `dist/manifest.json` 文件格式是否正确
2. 确认所有文件都已构建（运行 `npm run build`）
3. 尝试重新加载插件

### 问题 2：页面没有浮动面板
**症状**：访问 Amazon 页面后，没有看到浮动面板

**解决方案**：
1. 确认在正确的 Amazon 产品页面（URL 包含 `/dp/XXXXXXXXXX`）
2. 等待 2-3 秒，让脚本初始化
3. 按 F12 打开控制台，查看是否有错误信息
4. 尝试刷新页面

### 问题 3：数据显示为 NaN 或 0
**症状**：价格显示 $NaN，BSR 显示 #0

**解决方案**：
1. 检查控制台日志，查看是否有 `[Buyda选品助手]` 的输出
2. 确认 Amazon 页面已完全加载
3. 尝试访问其他产品页面
4. 查看控制台中的具体错误信息

### 问题 4：按钮点击无反应
**症状**：点击"保存到选品清单"或"计算利润"没有反应

**解决方案**：
1. 打开控制台，查看是否有错误信息
2. 确认 popup 功能是否正常
3. 尝试重新加载插件

## 🔄 自动化测试脚本（可选）

如果要批量测试多个产品，可以创建一个简单的自动化脚本：

```javascript
// 在浏览器控制台运行
const testProducts = [
  'https://www.amazon.com/dp/B07SQZWFTC',
  'https://www.amazon.com/dp/B08N5KWB9H',
  // 添加更多产品链接
];

testProducts.forEach((url, index) => {
  setTimeout(() => {
    window.open(url, `_blank_${index}`);
  }, index * 5000); // 每 5 秒打开一个新标签
});
```
