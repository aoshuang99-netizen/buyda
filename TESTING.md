# AI 选品插件 - 测试指南

## 快速开始

### 1. 加载插件到 Chrome

1. 打开 Chrome 浏览器
2. 访问 `chrome://extensions/`
3. 右上角开启 **"开发者模式"**
4. 点击 **"加载已解压的扩展程序"**
5. 选择项目文件夹：`/Users/frankzhao/WorkBuddy/20260327160327/ai-selection-extension/dist`
6. 插件将出现在扩展列表中 ✅

### 2. 测试核心功能

#### 测试 Popup 分析功能

1. 访问任意亚马逊产品页面，例如：
   - https://www.amazon.com/dp/B08N5KWB9H
   - https://www.amazon.com/dp/B07XJ8C8F5

2. 点击浏览器右上角的 **AI 选品助手** 图标 🤖

3. 点击 **"开始分析"** 按钮

4. 验证数据提取：
   - ✅ 产品标题
   - ✅ 价格
   - ✅ BSR 排名
   - ✅ 评分
   - ✅ 评论数
   - ✅ 品牌
   - ✅ 库存状态
   - ✅ Prime 资格

#### 测试浮动面板功能

1. 在亚马逊产品页面刷新页面
2. 右侧会自动显示 **AI 选品分析** 浮动面板
3. 验证显示：
   - ✅ 产品基本信息
   - ✅ 核心指标卡片（价格、排名、评分、评论）
   - ✅ AI 快速评估（综合得分、潜力等级）
   - ✅ 类别、库存、Prime 信息
   - ✅ 操作按钮（保存到清单、计算利润）

#### 测试利润计算功能

1. 在 Popup 中切换到 **"利润"** 标签
2. 输入以下测试数据：
   - 工厂成本：¥15

   - 广告费：$4
3. 点击 **"计算利润"**
4. 验证结果：

   - ✅ 利润率百分比

#### 测试监控清单功能

1. 在分析页面点击 **"保存到监控清单"**
2. 切换到 **"监控"** 标签
3. 验证：
   - ✅ 产品出现在清单中
   - ✅ 显示保存时间
   - ✅ 可以删除产品
   - ✅ 监控数字更新

#### 测试 AI 推荐功能

1. 切换到 **"AI推荐"** 标签
2. 验证显示 3 个示例推荐产品
3. 验证每个产品显示：
   - ✅ 产品标题
   - ✅ 价格
   - ✅ 潜力等级（高/中/低）
   - ✅ 推荐理由
   - ✅ 亚马逊链接

#### 测试设置功能

1. 切换到 **"设置"** 标签
2. 输入 Keepa API Key（可选）
3. 点击 **"保存"**
4. 验证 API Key 已保存（重新打开设置时显示）

### 3. 测试场景

#### 场景 1：高潜力产品分析

访问高评分、高销量产品：
- 评分 ≥ 4.5
- 评论数 ≥ 500
- BSR 排名 ≤ 10000

- AI 快速评估显示 **"高潜力"**（绿色）
- 综合得分 ≥ 75

访问中等销量产品：
- 评分 3.5-4.0
- 评论数 100-500
- BSR 排名 10000-50000

**预期结果：**
- AI 快速评估显示 **"中等潜力"**（黄色）
- 综合得分 50-74

访问低销量或无库存产品：
- 评分 < 3.5
- 评论数 < 100
- BSR 排名 > 50000
- 无库存

**预期结果：**
- AI 快速评估显示 **"低潜力"**（红色）
- 综合得分 < 50

### 4. 常见问题

#### Q: 插件加载失败
- 确保 `dist` 文件夹完整
- 检查是否开启了开发者模式
- 查看扩展页面是否有错误提示

#### Q: 无法提取产品数据
- 确保在亚马逊产品详情页（不是搜索结果页）
- 刷新页面
- 检查 URL 是否包含 `/dp/ASIN` 格式

#### Q: 浮动面板不显示
- 刷新亚马逊页面
- 检查扩展是否有权限访问该页面
- 查看浏览器控制台是否有错误

#### Q: 数据不准确
- 亚马逊页面结构可能变化
- 需要更新选择器
- 不同站点（.com vs .cn）可能有差异

### 5. 调试技巧

#### 查看扩展日志

1. 访问 `chrome://extensions/`
2. 找到 AI 选品助手
3. 点击 **"检查视图"** → **"背景页"** 或 **"弹出页"**
4. 在开发者工具 Console 中查看日志

#### 查看内容脚本日志

1. 在亚马逊页面按 F12 打开开发者工具
2. 切换到 Console 标签
3. 查找 `提取产品数据` 相关日志

#### 重新加载插件

- 在 `chrome://extensions/` 页面
- 点击插件的 **刷新图标** 🔄
- 或者按 `Ctrl+R` (Windows) / `Cmd+R` (Mac)

### 6. 性能测试

#### 测试加载时间

1. 关闭所有标签页
2. 打开一个亚马逊产品页面
3. 记录浮动面板出现时间
4. 应在 2-3 秒内显示

#### 测试内存占用

1. 打开任务管理器（`Shift+Esc`）
2. 查看 AI 选品助手的内存占用
3. 应在 20-50MB 范围内
