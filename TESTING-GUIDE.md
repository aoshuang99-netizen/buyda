## ✅ 第一步：验证插件已加载

1. 打开 Chrome 扩展页面：`chrome://extensions/`
2. 找到"AI选品助手"卡片
3. 确认开关是打开状态（蓝色）

## 🔧 第三步：测试 Popup 功能

1. 点击浏览器右上角的插件图标
2. 验证四个标签页：

### 标签页 1：分析
- 显示当前页面的产品数据
- AI 评估结果
- 快捷操作按钮

### 标签页 2：利润
- 输入工厂成本、运费、FBA 费用、广告费
- 自动计算毛利和利润率
- 颜色编码（绿色 >20%，黄色 10-20%，红色 <10%）

### 标签页 3：AI 推荐
- 显示示例推荐
- 1688 货源链接（待集成）

### 标签页 4：监控
- 查看已保存的产品列表
- 删除功能
- 保存时间显示

### 标签页 5：设置
- Keepa API Key 输入框（待集成）

## 🔍 调试技巧

### 查看错误日志
1. 右键点击插件图标 → 检查弹出内容
2. 或在 Amazon 页面：右键 → 检查 → Console 标签

### 重新加载插件
1. `chrome://extensions/` 页面
2. 找到"AI选品助手"
3. 点击刷新按钮 🔄

### 清除存储数据
```javascript
// 在 Console 中执行
chrome.storage.local.clear(() => {
  console.log('存储已清除');
});
```
